Técnico Desenvolvimento de Sistemas

No momento me encontro finalizando um curso técnico de Desenvolvimento de Sistemas (DS) que foi iniciado em 2023 no SESI. Alguns dos meus projetos se remetem à minha competência de desenvolvedor de sistemas, direcionando cada aluno a realizar estudos, desenvolver autonomia e trabalhos para o ramo profissional. Acompanhe abaixo alguns deles:

Boss Bridge - Startup

Uma startup em equipe com foco no desenvolvimento de um aplicativo para investidores e empresas, onde possam ter uma prestação de serviços em nosso sistema. Foco em autonomia e em facilitar conexão, comunicação e transferências de capital entre empresas e seus investidores.

Code Ware - IA IDE

Um projeto particular de um sistema de IDE com IA 100% nacional, com o propósito de auxiliar desenvolvedores iniciantes e avançados, oferecendo conforto visual e facilidade de acesso, com baixa taxa de cobrança de uso.

Sistema Bibliotecário

Um aplicativo feito em Python para o funcionamento de um sistema de biblioteca/livrarias escolares onde é permitido o registro de livros, fotos de capa, nome de autor, registro de empréstimo, data de devolução e alunos vinculados.

App Mobile Microvendas

Um aplicativo de celular que funciona como um registro de produtos e vendas, facilitando o trabalho diário dos pequenos vendedores de lojas e comércio. Banco de dados SQL integrado e fácil registro de contas e login.

Boss Bridge

Uma startup feita em equipe e iniciada como um projeto do meu curso técnico em DS no SESI Ibura. Temos o propósito de oferecer prestação de serviços a investidores e empresas, funcionando como uma ponte entre eles, facilitando o trabalho e a conexão entre empresa e empreendedor.

Code Ware

Uma IDE para programação ainda em desenvolvimento. A ideia é ser um sistema que funcione com uma IA para auxiliar na manutenção e geração de códigos. Totalmente brasileiro e com design responsivo, garantindo conforto visual para os desenvolvedores.

App Biblioteca

Um aplicativo com banco de dados integrado que funciona como um registro de livros e empréstimos para alunos em uma instituição acadêmica. Interface simples e eficiente para administração de bibliotecas escolares.

App Vendas

Aplicativo mobile para pequenos comerciantes que desejam utilizar um sistema simples e funcional. Permite registro de produtos, vendas, preços e cálculo automático de lucros, com armazenamento por data e hora.
